# Age Calculator using JSP & Servlet

This is a simple web-based Age Calculator application developed using JSP, Servlet, and Java. The user enters their Birth year and parsent year, and the system calculates the current age using the logic parsent in servlet. The request is sent through JSP, processed by a Servlet, and the final result is displayed back on a JSP page.
